# booking-page-mobile-responsive-UI-simplification
📅 Advanced Booking Page

A responsive booking page built with HTML, CSS, and JavaScript that allows users to schedule appointments with dynamic pricing, add-ons, promo codes, and real-time booking summaries.

✨ Features

- Mobile-friendly responsive UI
- Appointment scheduling form
- Dynamic price calculation
- Promo code discounts
- Add-on services
- Real-time booking summary
- Date validation
- Success notification

🛠️ Tech Stack

- HTML5
- CSS3
- JavaScript

🚀 How to Run

1. Clone the repository.
2. Open "index.html" in your browser.
